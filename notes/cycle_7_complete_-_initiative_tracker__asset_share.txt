## Architecture Ledger Update

### Combat Flow Diagram
```
DM pushes ⚔️ → HUD appears
  → Create Encounter
    → Add Combatants (type name/manual or import from tokens)
      → Set initiatives (sortable input fields)
        → Start Combat (sorts by initiative descending)
          → [Active Turn] Current combatant highlighted with timer
            → Damage/Heal via quick-damage inputs on hover
            → Status effect toggles (9 conditions)
            → Next Turn / Previous Turn
              → Round tracking (auto-increment)
                → End Encounter
```

### Asset Share Flow Diagram
```
DM opens 📦 panel
  → Selects image from library
    → Sets label + optional note
      → Pushes to all players
        → Players see asset viewer overlay
          → DM clicks "Deposit to [Player]"
            → Player sees Accept/Decline prompt
              → Accepted: Item added to inventory
```
