## Components Built

### PlayerCharacterSheet (Basic)
- `CharacterHeader` — Name, race, class, level, XP
- `CharacterQuickStats` — Ability scores, saves, skills at a glance
- `CharacterHpEditor` — Inline HP editing with temp HP support
- `CharacterXpEditor` — XP editing
- `RestControls` — Short/Long rest with resource recharge
- `ResourcesSection` — Class resources (Ki, Bardic Inspiration, etc.)
- `ResourceEditor` — Per-resource editing
- `SpellSlotsDisplay` — All 9 spell levels with used/max gauges
- `InventoryEditor` — Full inventory management
- `CurrencyEditor` — PP/GP/EP/SP/CP editing
- `DeathSaveTracker` — Visual death save tracking
- `SpellcastingSheet` — Full spell list with prepared status

### PlayerCharacterSheetPremium
- `PortraitSection` — Avatar with class icon, level badge, race/class/player name
- `PrimaryStatsRow` — AC, Initiative, Proficiency, Speed, Hit Dice, Passive Perception
- `HpBarSection` — Animated HP bar, current/max/temp, edit button, death state indicator
- `XpProgressSection` — XP bar with threshold markers, next level at indicator
- `AbilityScoreGrid` — 6 circular ability gauges with mod, saving throw, proficiency
- `SavingThrowsSection` — All 6 saves with bonuses
- `SkillsSection` — All 18 skills with modifiers, proficient/expertise indicators
- `WeaponsSection` — Computed attacks from equipment: name, bonus, damage dice, type, properties
- `SpellcastingSection` — Spell save DC, ATK bonus, cantrips, spell slots with gauges
- `ResourcesSection` — Custom class resources with recharge
- `InventorySection` — Equipment items + inventory items  
- `CurrencySection` — 5 coin types with edit
- `FeaturesSection` — Racial, class, and feat features
- `RestAndLevelSection` — Short rest, long rest, level up buttons
- `DeathSaveTrackerSection` — Success/failure tracking
- `BackstorySection` — Character biography
- `SpeedBreakdown` — Walk/fly/swim/climb/burrow speeds
- `ConditionsSection` — Active condition badges

### Combat Math Engine
- Weapon DB with 20+ weapons: dice, damage type, properties (finesse, versatile, light, heavy, thrown, ranged)
- `computeAttacks()` — Auto-derived from equipment inventory using STR/DEX + proficiency
- Proficiency bonus: `ceil(1 + level/4)`
- Ability mod: `floor((score - 10) / 2)`

### Real-Time Firebase Sync  
- `usePlayerFirebaseSync` subscribes to: sessions, campaign meta, combat log, combatants, homebrew items/spells/feats
- All updates via `onSnapshot` — no polling, no refresh needed
- Combatant HP sync: DM edits HP → player sees instantly
